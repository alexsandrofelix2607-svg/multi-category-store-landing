# Multi Category Store Landing 🛍️

Landing page responsiva com busca e filtro por categoria. Digite "Cafeteria" ou "Mercado" para filtrar produtos. 
para lojas com múltiplas categorias. 

## 📌 Sobre o Projeto
Este site foi criado para servir como modelo de landing page para comércios como: 
Cafeteria, Mercado, etc. 
O foco é ser 100% responsivo e moderno.

- ✅ Design totalmente responsivo
- ✅ Templates por categoria
- ✅ Layout moderno e limpo
- ✅ Otimizado para Mobile
